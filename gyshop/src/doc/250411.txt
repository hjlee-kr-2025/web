DAO : (Data Access Object)
 - 데이터를 보관하고 있는 곳 : DB 서버
 - DAO는 데이터베이스를 access 하도록 구현했습니다.
 - 하나의 쿼리가 하나의 메서드를 사용하도록 만들었습니다.
 - 하나의 메서드는 7단계의 동작을 거칩니다.
 (1. 드라이버확인, 2. DB연결, 3. SQL작성, 4. 실행객체세팅
   5. 실행 및 결과리턴  6. 결과저장, 7. DB닫기)
   
VO : (Value Object)
 - 모듈(기능)에서 사용할 데이터를 보관합니다.
